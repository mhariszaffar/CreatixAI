# Creatix AI - Vyrothon (Stage 1) Hackathon 2026

An AI-first productivity platform designed for maximum product education with minimum drop-off.
This repository contains my individual UI/UX design submission for the Stage 1 'Creatix-AI' challenge at the Vyrothon hackathon (hosted at NUST). This submission successfully qualified me to represent GIKI in the Stage 2 cross-functional team MVP build.

## 📌 Project Overview
The Creatix AI challenge tested rapid UI/UX problem-solving, adherence to strict design constraints, and the ability to justify design decisions through testing. 

While I initially explored using React and Framer for an interactive prototype, the hackathon guidelines strictly mandated a native Figma prototype submitted via a public view link. 

## 📂 Deliverables & File Structure
The project is cleanly organized into four primary user flows, alongside comprehensive documentation:

* **Cover Page:** A high-level summary of the design approach and core problem-solving strategy.
* **01 Auth:** Secure and frictionless sign-up/login flows.
* **02 Onboarding:** Low-cognitive-load user setup screens.
* **03 Dashboard:** The primary user interface and data visualization hub.
* **04 Style Guide:** A basic, scalable design system encompassing typography, color palettes, and component states.
* **A/B Test Write-up:** A detailed breakdown of design iterations, testing methodologies, and data-backed decisions.
